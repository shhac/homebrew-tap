# Homebrew Tap

Personal Homebrew tap for distributing custom CLI tools.

## Installation

```bash
brew tap shhac/tap
```

## Available Formulas

| Formula | Description | Version |
|---------|-------------|---------|
| [toss](https://github.com/shhac/toss) | Fast dice rolling CLI with colored output and reproducible rolls | 0.5.1 |
| [git-wt](https://github.com/shhac/git-wt) | Fast CLI tool for managing git worktrees with enhanced features | 0.4.3 |

## Usage

Install a formula:

```bash
brew install shhac/tap/toss
brew install shhac/tap/git-wt
```

Or after tapping:

```bash
brew install toss
brew install git-wt
```

## Platform Support

All formulas support:
- macOS (Apple Silicon & Intel)
- Linux (ARM64 & x86_64)
